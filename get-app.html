📥 Histatu Runner

The in-game companion app: press your open key on a chest and it's logged to the shared map — plus route racing (a stopwatch that counts your opens against a route), cooldown tracking, and leaderboards.

🪟 Windows

zip — no install, no Python
⬇ Download for Windows
  1. Unzip HistatuRunner-windows.zip anywhere, then run HistatuRunner.exe inside the HistatuRunner folder (keep the folder together — the exe needs the files beside it).
  2. First launch shows “Windows protected your PC” — that's SmartScreen warning about a new unsigned app, not malware. Click More info → Run anyway (one time only).
  3. In game, press F7 until the WORLD debug panel shows, enter your IGN in the app, and open chests.
A plain folder of files, not a self-extracting installer — that's deliberate, it trips far fewer antivirus false positives. Nothing ever leaves your PC unless you contribute it to the shared map; it never captures or uploads screenshots. To update, download the new zip and unzip it over your folder (your capture_config.json settings stay).

🐧 Linux

source zip — Python 3.10+, X11 & Wayland
⬇ Download for Linux
  1. Unzip, then run ./run.sh — it checks and installs what's needed for your display server.
  2. X11: pip install pillow pytesseract pynput + the tesseract-ocr package (optional xdotool for docking).
  3. Wayland (Hyprland/Sway): grim + pip install evdev, and add yourself to the input group: sudo usermod -aG input "$USER" (log out/in).
No prebuilt Linux binary — it runs from source with a tiny dependency set.
What it does: logs every chest you find to the shared map (or submits it for review), records loot routes and times your runs of them, learns real travel times between chests, tracks the daily 8 PM Eastern chest reset per player, and keeps route leaderboards — all synced with the shared dungeon map. It reads only your own screen (the F7 panel) — no mods, no injection, nothing touches the game — and checks for updates on launch.
🔓 Fully open source — don't take our word for any of this. The complete source for the app, this website, and the server is public at github.com/blakebiz-dev/Histatu-Dungeon-Companion (MIT licensed). The downloads above are built automatically by GitHub Actions from that exact tagged source — you can read every line the app runs (it's one Python file), audit the build workflow, or build it yourself.

What it never does: no game mods, no injection, no memory reading, no packet sniffing, no game-file access. It works entirely by reading the F7 debug panel off your own screen with OCR, and the only things it shares are what you contribute to the community map: chest locations, your opens (for your cooldown tracking), travel times, and route completion times. The README walks through the whole security model.
