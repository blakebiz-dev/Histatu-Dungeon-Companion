💾 Backup & restore

Export downloads the entire shared map — chests, groups, mobs, routes, chest types, mob categories, calibration, learned travel times, leaderboards and player logs — as one JSON file.

Restore

Uploads the map structure (chests, groups, mobs, routes, chest types, mob categories) from a backup file to the shared map for everyone. Existing entries with the same id are overwritten; nothing is deleted. Large maps upload gradually (the shared store limits writes to ~30/min).

Maintenance

When a chest is removed, the travel times learned to it stay behind. Left alone they eventually fill the store and block new measurements. Clean up drops every travel time whose chest is gone (both are re-learned automatically as routes get run).

🗑 Recycle bin

Markers removed in the last 7 days. Restore puts them back on the map for everyone.

❓ Guide

This map is a shared, community chest & route tracker for the dungeon. Anyone can browse it and track their own cooldowns; editors (with a write key) maintain the pins. The companion app reads the in-game F7 panel to log chests and time your route runs.

Map tools — the pill buttons top-left

✋ Select
Pan, zoom, and click any pin to inspect it (or, as an editor, edit it). The default tool.
📦 Chest Editor
Click the map to place a single chest, or open one and enter its in-game X/Y/Z.
⬡ Group Editor
One pin standing for a cluster of chests at a spot — you set how many. The route generator treats it as that whole pack.
☠ Mob Editor
Mark a mob spawn (XP, spawn count, category) — handy reference, excluded from loot routes.
🌀 Teleport Editor
Mark a teleport/waypoint spot so runners know the fast ways around.
🧭 Route Editor
Build a named route by clicking its chests in order; publish it so others can run and race it.
◈ Reliability grade
Each route shows an A–F grade for how trustworthy it still is: how recently its chests were actually opened by anyone (proof they still exist), plus how well-measured its leg times are. A ⚠ flags a route whose stops were removed from the map or whose loot hasn't been confirmed in a long time.

View & find

Chests · Groups · Mobs · Teleports · Routes
Show/hide each kind of pin to de-clutter the map.
👤 IGN
Type a player name to dim the chests they've already opened this reset (cooldown) — so you see only what's still up for them.

Jump to a chest by its in-game coordinates — type X Z (or X Y Z) and press Enter.
🌡 Coverage
Colour each area by how well its chest-to-chest travel times are measured — red = scout here to sharpen routes, green = well-known.
⤢ Fit · ＋ －
Fit the whole map to the screen, or zoom in/out (mouse wheel works too).

Stats & leaderboards — 🏁 Runs on the toolbar, the rest under ⋯ More

🏁 Runs
Fastest published-route times. Players tied for a route's record all share 🥇 (co-holders), and each route shows a typical time averaged over every run — a realistic target, not just the record.
📈 My stats
Your chests/hour, session history and personal bests (from opens logged in the app).
🏆 Contributors
Who has found and verified the most chests.

Under ⋯ More — maintenance & editor tools (most need a write key)

🔑 Editor
Paste your write key to unlock editing. Without it the map is view-only (you can still track your own cooldowns and run routes).
🎯 Calibrate Editor
Line up in-game X/Z with the map image — required before coordinate-based placement and before the app can draw pins. Calibrating never moves existing pins — after a map-image swap you'll be offered ⤾ Re-align to snap them over.
⤾ Re-align pins Editor
After swapping the map image and re-calibrating, snaps every pin back to its saved in-game coordinates — and refits the drawn area outlines onto the new image the same way. If pins ever look uniformly shifted, this is the fix (the tool warns you when it detects that).
▱ Draw area Editor
Outline a named region (button in the sidebar); chests inside it count toward that area's completion total.
🔎 Verify Editor
On the toolbar: confirm or reject the chest locations and removals players have submitted.
🗑 Bin Editor
Restore any marker removed in the last 7 days.
💾 Backup
Export the whole map as a JSON file (anyone), or restore it and clean up orphaned travel data (editors).
🧹 Clean up Editor
In Backup: prune travel times measured to chests that no longer exist, freeing room for new ones. Find it under Backup → Maintenance.

Reading the pins

Chest / cluster
A gold square is one chest; a dashed gold circle is several chests too close to separate — click to expand.
Group size
Group pins are coloured by pack size: red 1–3, orange 4–7, purple 8+.
Dimmed pin
Greyed out = on cooldown for the player in the 👤 box (opened since the last 8 PM ET reset).
Gold-ringed pin
Part of the route you're previewing, numbered in run order.

The companion app — logs chests & times your runs

Modes
⏺ Log add/record chests you open · 🧭 Record capture a route's stops & leg times · 🔎 Verify (editors) confirm submissions · ▶ Run race a route against the clock.
During a run
A stopwatch counts your opens against the route's stops and records your finish time to the leaderboard. It ⏸ auto-pauses in menus, on lost focus, or when you stand still, so a break doesn't ruin your time. Study the route here on the map first — navigating it is the skill.
♻ Reset cooldowns
Tap it after the random in-game event that unlocks every chest early, so the map shows them as available again.
